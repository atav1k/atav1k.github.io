Project Goals

Building a secure digital art marketplace is hard that when it comes to the user discovering new art and displaying it, you want it to be simple and enjoyable.

How do you design a web and mobile app that adhere to existing visual art UI paradigms so a first time user is just a few clicks away from displaying HD artwork on their LCD screen.

My contact was to deliver iOS mobile flows and a web frontend framework update, to drive user engagement and brand presence.

Information architecture of an iOS app

Architecting iOS screens and flows

After surveying existing features and desired functionality, I began with workflows to account for requirements of a commerce and gallery platform. The information architecture defined the mobile product and it evolved overtime, as opportunities and requirements changed.

I designed the iOS experience around tabbed navigation to feature a few core experiences, and conducted rapid testing using an Xcode clickable prototype. The product design was iterated until we’d arrived at an achievable first release in the app store. Streaming displays to Apple Displays and Chromecast offered some design challenges that were solved using native iOS actions and helper screens.

Xcode prototype screenshot

Refreshing the web application

The main goal was to bring consistency to the web app. I worked with a small team to build out a core set of components and screens in Bourbon which offered many design patterns baked in.

After a few days of mood boarding and sketching , we designed and coded the frontend framework with an updated brand identity and new screen layouts. We’d was designing a responsive frontend where the artwork displayed well across all devices and viewports. We also componentized UI elements, which required surveying the application and forming core HTML/CSS atoms and molecules  according to from Brad Frost’s Atomic Design principles.

Depict launched a new iOS app and a refreshed web application in late summer of 2014 after two months of design and development, leading up to the launch of their 4K screen hardware.
